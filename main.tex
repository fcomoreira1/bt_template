\subsection{Guidelines for your thesis report}

\begin{itemize}
\item Structure your section, subsection, paragraph as you like, but avoid the use of chapters.
%
\item Use bibtex to cite related work, e.g., \cite{DBLP:journals/x/Turing37}, adding your reference in the {main.bib} file.
%
\item Change the value of the commands {$\backslash$thesistitle}, {$\backslash$authorname}, {$\backslash$supervisor}, {$\backslash$supervisorinstitution}, {$\backslash$abstracttext} to set up the content of the title page. These commands are beginning of the {main.tex} file.
%
\item The thesis must be \textbf{at most} 30 pages long and \textbf{at least} 20 pages long.
%
\item Shoot for quality and not or length: we prefer reading 20 pages of high quality, well explained, work rather than 30 pages of messy content. 
%
\item The appendix is \textbf{not included} in the page count (\textbf{WARNING:} the title page, abstract page, table of content page, and all the bibliography instead are included in the page count!).
%
\item Note: we will not evaluate your appendix, which should not contain your thesis contribution but only provide additional details (e.g., some code listing if they are important, additional tables for the experiments, \ldots).
%
\item You \textbf{cannot} change the tex template (e.g., font size, margin size, \ldots) to gain more space.
\end{itemize}
